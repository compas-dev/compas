A package for numerical computation.

Core

geometry

plot_scalarfield_contours(xy, s[, N]) Plot the contours of a scalarfield.

linalg

nullspace(A[, tol]) Calculates the nullspace of the input matrix A.
rank(A[, tol]) Calculates the rank of the input matrix A.
dof(A[, tol, condition]) Returns the degrees-of-freedom of the input matrix A.
pivots(U[, tol]) Identify the pivots of input matrix U.
nonpivots(U[, tol]) Identify the non-pivots of input matrix U.
rref(A[, algo, tol]) Reduced row-echelon form of matrix A.
chofactor(\*args) Returns the Cholesky factorisation/decomposition matrix.
lufactorized(\*args) Return a function for solving a sparse linear system (LU decomposition).
normrow(A) Calculates the 2-norm of each row of matrix A.
normalizerow(A[, do_nan_to_num]) Normalise the rows of matrix A.
rot90(vectors, axes) Rotate an array of vectors through 90 degrees around an array of axes.
solve_with_known(A, b, x, known) Solve a system of linear equations with part of solution known.
spsolve_with_known(A, b, x, known) Solve (sparse) a system of linear equations with part of solution known.
