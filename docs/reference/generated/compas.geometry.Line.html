Line

class compas.geometry.Line(p1, p2)[source]

Bases: object

A line object is defined by two points in three-dimensional space.

Parameters:

  • p1 (tuple, list, Point) – The xyz coordinates of the first point.

  • p2 (tuple, list, Point) – The xyz coordinates of the second point.
start

Point – The start point.

end

Point – The end point.

length

float – (read-only) The length of the line between start and end.

midpoint

Point – (read-only) The midpoint between start and end.

direction

Vector – (read-only) A unit vector pointing from start to end.

Note

For convenience, this class implements the following magic methods:

  • __repr__

  • __len__

  • __getitem__

  • __setitem__

  • __iter__

  • __mul__

  • __imul__

Examples

>>> line = Line([0,0,0], [1,1,1])
>>> line.midpoint
[0.5, 0.5, 0.0]
>>> line.length
1.73205080757
>>> line.direction
[0.57735026919, 0.57735026919, 0.57735026919]

>>> type(line.start)
<class 'point.Point'>
>>> type(line.midpoint)
<class 'point.Point'>
>>> type(line.direction)
<class 'vector.Vector'>
