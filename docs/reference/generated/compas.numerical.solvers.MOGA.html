MOGA

class compas.numerical.solvers.MOGA[source]

This class contains a binary coded, multiple objective genetic algorithm called NSGA-II (ref to K. Deb). NSGA-II uses the concept of non-domination, or Pareto-domination to classify solutions and optimize as a genetic algorith. NSGA-II also employs a crowding distance operator especially designed to distribute individuals in the population allong the Pareto front, and this avoid crowding in small areas. The main function is MOGA.moga, calling this starts the multi objective optimization process.
