geometric_key

compas.utilities.geometric_key(xyz, precision='3f', tolerance=1e-09, sanitize=True)[source]

Convert XYZ coordinates to a string that can be used as a dict key.

Parameters:
  • xyz (sequence of float) – The XYZ coordinates.
  • precision (str) – Optional. A formatting option that specifies the precision of the individual numbers in the string. Supported values are any float precision, or decimal integer ('d'). Default is '3f'.
  • tolerance (float) – Optional. A tolerance for values that should be considered zero. Default is 1e-9.
  • sanitize (bool) – Optional. Flag that indicates whether or not the input should be cleaned up using the tolerance value. Default is True.
Returns:

str – the string representation of the given coordinates.

Example

from math import pi
from compas.utilities import geometric_key

print(geometric_key([pi, pi / 2.0, 2.0 * pi], '3f'))
