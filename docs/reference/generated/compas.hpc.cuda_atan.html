cuda_atan

compas.hpc.cuda_atan(a)[source]

Trigonometric arctangent of GPUArray elements.

Parameters:a (gpu) – GPUArray with elements to be operated on.
Returns:gpu – atan(GPUArray)

Examples

>>> a = cuda_atan(cuda_give([0.5, 1]))
array([ 0.46364761,  0.78539816])
>>> type(a)
<class 'pycuda.gpuarray.GPUArray'>
