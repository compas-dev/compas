Mesh.get_faces_attribute

Mesh.get_faces_attribute(name, value=None, fkeys=None)

Get the value of a named attribute of multiple faces.

Parameters:
  • name (str) – The name of the attribute.
  • value (object (None)) – The default value.
  • keys (list (None)) – A list of identifiers. Defaults to all faces.
Returns:

values (list) – A list of values of the named attribute of the specified faces.
