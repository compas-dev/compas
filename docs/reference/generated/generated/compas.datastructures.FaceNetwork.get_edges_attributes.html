FaceNetwork.get_edges_attributes

FaceNetwork.get_edges_attributes(names, values=None, keys=None)

Get the values of multiple named attribute of multiple edges.

Parameters:
  • names (list) – The names of the attributes.
  • values (list (None)) – A list of default values. Defaults to a list of None s.
  • keys (iterable (None)) – A list of edge identifiers. Each edge identifier is a pair of vertex identifiers. Defaults to all edges.
Returns:

values (list of list) – The values of the attributes of the specified edges. If an attribute does not exist for a specific edge, it is replaced by the default value.
