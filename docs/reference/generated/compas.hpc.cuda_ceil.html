compas.hpc.cuda_ceil(a)[source]

Parameters:a (gpu) – GPUArray with elements to be operated on.
Returns:gpu – ceil(GPUArray)

Examples

>>> a = cuda_ceil(cuda_give([0.5, 0.1, 1.9]))
