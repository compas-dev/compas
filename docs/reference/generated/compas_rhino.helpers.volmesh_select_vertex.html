volmesh_select_vertex

compas_rhino.helpers.volmesh_select_vertex(volmesh)[source]

Select a vertex of a volmesh.

Parameters:volmesh (compas.datastructures.VolMesh) – A volmesh object.
Returns:
  • key (int, str, tuple, frozenset) – The identifier or key of the selected vertex.
  • None – If no vertex was selected.

Examples

>>> key = volmesh_select_vertex(volmesh)
