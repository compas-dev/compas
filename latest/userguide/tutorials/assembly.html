Assemblies

The Assembly data structure provides functionality for modelling and managing the connections between the individual components of an … assembly. Each component is either a part defined by geometrical objects, or another assembly.

Basic Usage

from compas.datastructures import Part
from compas.datastructures import Assembly
from compas.geometry import Box

assembly = Assembly()
